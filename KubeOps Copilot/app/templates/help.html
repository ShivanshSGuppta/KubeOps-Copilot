Help

KubeOps Copilot is a read-only Kubernetes operations assistant built for safe triage.

  • Mock mode (default): uses sample data so the app works without a cluster.
  • Real mode: connects to a Kubernetes cluster using your kubeconfig (still read-only).
Safety model
  • Only allowlisted, read-only actions are executed.
  • User input is sanitized and validated before any action runs.
  • Every NL query is written to an audit log.
Try these natural language queries
  • show namespaces
  • pods in kube-system
  • failing pods in default
  • events in kube-system
